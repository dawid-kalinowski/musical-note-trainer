An interactive web application for learning musical notes that generates random notes across octaves and instruments (guitar, bass and piano), with customizable practice settings and real-time feedback using plain Javascript, HTML and CSS.  

Available instruments:  
- Guitar  
- Bass guitar  
- Piano  
- Violin  

It can generate notes within ranges selected by user within natural notes, sharps (#), and flats (b). Users can also select custom notes and octaves to practice, or even predefined note ranges for specific instruments


Live version available here:  
https://dawid-kalinowski.github.io/musical-note-trainer/

The tuner itself is a fork of the following repository:  
https://github.com/qiuxiang/tuner
